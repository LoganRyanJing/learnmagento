github搜索技巧

  • github
  • 随笔

awsome

今天无意中在群里看到同伴在聊天，了解到github上的一个项目 awesome-clojure，打开发现，里面竟然汇集了大量clojure库。而后我继续深究，想知道awesome是个啥。发现只要在github上搜索加上awesome可以得到很多有用的东西。比如:

  • javascript awesome
  • java awesome
  • php awesome

cookbook，tutorial

在你打算学习一些类库或者一门语言时候，也可以加上cookbook,tutorial关键词进行搜索。
此外，这里还有更全面的免费编程教程列表：free-programming-books
